Calculator
This is a basic calculator web application built with HTML, CSS, and JavaScript. It allows users to perform simple arithmetic operations like addition, subtraction, multiplication, and division.

Features
Responsive layout for different screen sizes
Keyboard support for easier input
Error handling for invalid input
History of previous calculations
Getting Started
To use the calculator, simply open the index.html file in a web browser.

Usage
Enter the first operand using the number keys or by clicking on the calculator buttons.
Select the desired operation (+, -, x, /) by clicking on the corresponding button.
Enter the second operand using the number keys or by clicking on the calculator buttons.
Click on the "=" button to see the result.
To clear the input and start a new calculation, click on the "AC" button.
History
The history panel on the left side of the calculator displays a list of previous calculations. Each item in the list shows the previous operand, the operation, the current operand, and the result.
